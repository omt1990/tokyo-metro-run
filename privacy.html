プライバシーポリシー

制定日: 2026年4月20日 / バージョン 1.0

小俣勇祐（以下「運営者」）は、「東京メトロラン」（以下「本サービス」）におけるユーザーの個人情報の取り扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」）を定めます。

1. 基本方針

運営者は、ユーザーのプライバシーを尊重し、個人情報保護法その他の関連法令を遵守するとともに、本ポリシーに基づき個人情報の適切な取り扱いに努めます。

2. 取得する情報

本サービスは、以下の情報をユーザーから取得します。

取得情報取得方法任意/必須
プレイヤー名・チーム名ユーザーの入力必須
ゲームコード・参加履歴自動生成・ユーザー操作必須
ミッション達成の証明写真ユーザーのアップロード任意
駅訪問履歴・スコアゲーム進行に伴い自動記録必須
匿名認証 ID（UID）Firebase Authentication により自動発行必須
ブラウザの localStorage に保存されるセッション情報自動保存必須

本サービスは、氏名・住所・電話番号・メールアドレス・正確な位置情報（GPS）・支払情報を取得しません。

3. 利用目的

  1. 本サービスの提供・運営（ゲーム状態の共有、ランキング表示等）
  2. 不正利用の防止
  3. 本サービスの改善のための統計的分析（個人を特定しない形式）
  4. ユーザーからのお問い合わせへの対応

4. 第三者提供

運営者は、以下の場合を除き、取得した情報を第三者に提供しません。

  1. ユーザーの同意がある場合
  2. 法令に基づく場合
  3. 人の生命・身体・財産の保護のために必要がある場合

5. 業務委託先（データ処理を委託している事業者）

本サービスは以下の事業者のサービスを利用してデータを処理・保管しています。

事業者サービス用途所在地
Google LLCFirebase Authentication / Realtime Database / Hosting認証・データ保管・配信米国（データ保管リージョン: asia-southeast1 / シンガポール）

Google LLC のプライバシーポリシーは こちら をご参照ください。

6. データの保管期間

  1. ゲームデータは、運営者が削除するまで Firebase Realtime Database に保管されます。
  2. ユーザーは、後述「9. 開示・削除請求」に従い、自身のゲームデータの削除を請求できます。
  3. 運営者は、長期間利用されていないゲームデータを予告なく削除する場合があります。

7. Cookie・ローカルストレージ

  1. 本サービスは、ユーザーの利便性向上のため、ブラウザの localStorage を使用します。
  2. localStorage には、セッション情報（ゲームコード・プレイヤー ID）、同意済みの規約バージョン、一時的な写真キャッシュが保存されます。
  3. 本サービスは、現時点では Cookie および Google Analytics 等の外部アクセス解析ツールを使用していません。

8. セキュリティ

  1. 運営者は、取得した情報への不正アクセス・紛失・破壊・改ざん・漏洩を防止するため、Firebase Security Rules による認証済みユーザーへのアクセス制限等、合理的な安全管理措置を講じます。
  2. ただし、インターネットを経由する通信の完全な安全性は保証できないことを予めご了承ください。

9. 開示・削除請求

  1. ユーザーは、自身に関する情報の開示・訂正・削除を運営者に請求することができます。
  2. 請求は下記お問い合わせ先にメールで行ってください。
  3. 運営者は、請求を受領してから原則 2 週間以内に対応します。
  4. 請求の際、ユーザー本人であることを確認するため、ゲームコード・プレイヤー名等の情報提供をお願いする場合があります。

10. 未成年者の利用

未成年者が本サービスを利用する場合は、保護者の同意を得たうえで利用してください。保護者から未成年者のデータ削除の請求があった場合、運営者は速やかに対応します。

11. プライバシーポリシーの変更

  1. 運営者は、法令改正その他の事情により、本ポリシーを変更することがあります。
  2. 重要な変更の場合、本サービス上で告知するものとします。
  3. 変更後に本サービスを利用した場合、変更後のポリシーに同意したものとみなされます。

12. お問い合わせ

本ポリシーに関するお問い合わせ・開示請求・削除請求は、下記までご連絡ください。
